# 📊 SalesIQ Pro

Enterprise Sales Intelligence & Forecasting Platform built using Python, Streamlit, Plotly, Statsmodels, and SciPy.

## 🚀 Features

- Revenue Trend Analysis
- Sales Forecasting using SARIMA & Holt-Winters
- Anomaly Detection
- Monte Carlo Simulation
- Product & Region Analysis
- Strategic Business Insights
- Interactive Dashboard

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Statsmodels
- SciPy
